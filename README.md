# SeMIS v2 — 에어제타 보안종합정보시스템

기존 [kjsemis Google Sites](https://sites.google.com/view/kjsemis/)를 고도화한 Version 2.
정적 웹앱(HTML/CSS/JS)으로 어떤 웹서버(GitHub Pages 등)에서도 바로 호스팅 가능.

**접속 주소: https://mark4mission.github.io/semis-v2/** (GitHub Pages, `main` 브랜치 push 시 자동 반영)

## 구조

```
SeMIS_v2/
├── index.html       # 앱 셸 (로그인 모달, 헤더, 사이드바, 메인)
├── css/main.css     # 디자인 시스템 (반응형)
├── js/
│   ├── app.js       # 코어 엔진: 인증(SHA-256), 저장소+마이그레이션, 메뉴 엔진, 권한, 라우터
│   ├── modules.js   # 모듈: 대시보드(공지/보안등급), 시스템설정
│   ├── calendar.js  # 모듈: 일정관리 캘린더 (v2.1)
│   └── sync.js      # Supabase 공용 DB 실시간 동기화 (v2.1)
└── tests/
    └── run-tests.cjs # jsdom 테스트 스위트 (116건, CommonJS)
```

## 핵심 기능

- **로그인**: 모달에서 암호만 입력하는 간편 로그인. 암호는 SHA-256 해시로만 코드에 보관 (평문 미노출)
- **권한 3단계**: 시스템관리자(`mark3464`) / 보안관리자(`avsec`) / 일반사용자(`branch`)
- **메뉴 엔진**: 기존 사이트 전체 메뉴를 외부링크 메뉴로 이관 완료. 시스템 설정에서 웹주소 링크 메뉴 등록/수정/순서변경/권한설정 가능
- **대시보드**: 공지사항, 국가 항공보안등급 위젯(5단계, 시작~종료 기간 지정·만료 시 자동 복귀·적용일 예약·변경 이력), 보안점검 실적(계획 대비 완료율 + 이번 달 예정), CARES 환경센서(임계치 초과 강조 + 추이 그래프), 바로가기, 다가오는 일정. 카드 배치(v2.19): 좌 공지→환경센서→고장신고 / 우 등급→바로가기→일정→점검→만료→이수증
- **guest 경량 대시보드** (v2.19): 일반 사용자에게는 보안 카드 대신 🗞 보안 뉴스(Edge Function `semis-news`가 보안뉴스 boannews RSS + "항공/공항" 제목검색을 수집·분류, 30분 로컬 캐시) + 📊 항공보안 인사이트(보안등급 5단계·반입물품·검색절차·액체류 교육 시각화 탭) 중심의 화면 표시
- **보고체계 연락망** (v2.6): 기존 구글시트(8개 탭)를 랜딩페이지형 모듈로 내재화 — 30분 이내 SMS 보고 강조 배너, 구분별 보고 절차 아코디언, 사건별 보고처(①~④ 그룹), 기관별 연락처 카드(안전보안실·국토부·지방항공청 3청·비상안전기획관실·대테러/위기관리센터·TSA·서면보고 이메일), 전화/문자/메일 원터치 + 복사 버튼, 통합 검색(이름/기관/번호), 관리자 행 단위 편집(실시간 공유). **연락처 실데이터는 코드에 미포함** — 공용 DB(`semis_store` "contacts")에서만 동기화
- **지점 관리** (v2.7): 해외지점을 줌 가능한 세계지도(Leaflet) 위에 구역별 색상 마커로 표시 — hover 퀵 오버뷰, 클릭 상세 모달, 구역 필터·통합 검색, 목록 뷰. 입력: 구역(미주/유럽/중국/일본/아시아), 지점코드, IATA(주요 공항 좌표 자동 배치), 지점장/보안담당자/총원, 기내식, Layover 호텔, 주재 정비사 + 자유 추가 항목. CDN 차단 시 목록 자동 폴백
- **보안점검 일정관리** (v2.4, v2.6.1): 점검 결과를 시정조치/개선권고/현장시정/관찰사항 유형별 복수 등록(한 줄 요약), 목록 뷰 유형 배지 요약, 대시보드 유형별 통계 4칸 그리드. 구분(국내정기/불시평가/해외공항/주요일정)×월 연간 매트릭스 + 목록 뷰, 상태(계획/완료/연기/취소), 팀 점검관 태그, 일자 확정 시 캘린더 자동 연동, 2026 계획(구글시트) 시드 이관
- **CARES 연동** (v2.4, v2.6.1): 환경센서를 대시보드에 표시 — 기본은 임계치 초과 지표만(전체 정상 시 한 줄), "전체 표시" 토글로 8개 지표 확장. 무인증 공개 조회, 1분 자동 갱신. 연동 API 키는 코드에 두지 않고 공용 DB(`caresCfg`)에서 런타임 로드 (v2.6)
- **공지사항 리치 에디터** (v2.3): 서식(굵게/밑줄/목록), 표 삽입, 이미지 삽입(자동 축소 후 공용 저장소 업로드), 파일 첨부(10MB, Supabase Storage), HTML 살균 처리
- **일정관리 캘린더** (v2.1~v2.5):
  - 시작일-종료일 기간 일정(구글캘린더식 **한 줄 연결 바** 표시), 종일/시간 구분
  - 시간 일정은 투명 배경 + 색 점·색 글씨로 구분 (구글캘린더 스타일)
  - **반복 일정**: 매일/매주/2주마다/매월/매년 + 반복 종료일, 리마인더·ICS(RRULE) 연동
  - 메모에 링크·이미지·파일 첨부 (붙여넣기/드래그앤드롭 지원)
  - 14색 고대비 팔레트, 완료 체크(칩에서 바로 토글), 🚗 차량 예약 / 🏢 회의실 예약 체크
  - 리마인더: 2주일 전 / 1주일 전 / 1일 전 / 1시간 전 (반복 일정은 다음 occurrence 기준)
  - 담당자 카테고리: 팀원 태그(🧭박철성 · 🛡️최상일 · ✈️이은우 · 🌸이윤민) + 자유 입력, 필터
  - 보기 전환: 일 / 주 / 2주 / 월 / 년 (보기 상태 자동 저장)
  - 드래그앤드롭으로 일정 이동, ⇥ 핸들 드래그로 기간 조정

## 데이터 저장 (v2.1 — 공용 DB 실시간 공유)

- **Supabase 공용 DB** (`semis-v2` 프로젝트, 서울 리전, 테이블 `semis_store`)에
  메뉴·공지·일정·보안등급이력·사용자설정이 실시간 동기화됨 → 다중 사용자 공유
- **오프라인 폴백**: 네트워크 단절 시 localStorage로 계속 동작, 변경분은 대기열에 보관 후 재연결 시 자동 업로드
- **실시간 반영**: 다른 사용자의 변경이 Supabase Realtime으로 즉시 반영 (CDN 차단 시 30초 폴링 폴백)
- 헤더의 동기화 배지(🟢 실시간 / 🟡 동기화 중 / 🔴 오프라인)로 상태 확인, 클릭 시 수동 동기화
- 백업 파일 복원 시 복원본이 공용 DB에 우선 반영됨(forcePush)

## 초기 계정

초기 암호는 별도 전달(보안상 문서 미기재). 로그인 후 시스템 설정 → 사용자/암호에서 변경 권장.
암호 변경 내역·추가 사용자는 해시로만 저장되며 공용 DB를 통해 전 기기에 공유됨.
